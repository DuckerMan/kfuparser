# Документация

### Инициализация
```php
$parser = new Parser();
```

### Функции:

**Получение токена:** ```php
$token = $parser->auth('логин', 'пароль');```



**Получение баллов:** ```php
$stats = $parser->parse_menu_7('токен');```


**Получение информации о студенте:** ```php
$info = $parser->parse_student('токен'); ```
